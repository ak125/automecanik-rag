# Questions fréquentes - Paiement

## Quels moyens de paiement acceptez-vous ?

### Cartes bancaires
- Visa
- Mastercard
- Carte Bleue
- American Express

### Paiement en plusieurs fois
- **3x sans frais** : À partir de 100€ d'achat
- **4x sans frais** : À partir de 200€ d'achat
- Via Alma ou PayPal

### Autres moyens
- PayPal
- Virement bancaire (sur devis)
- Chèque (sur devis, délai d'encaissement)

## Le paiement est-il sécurisé ?

**Oui**, notre site utilise :
- Protocole HTTPS (chiffrement SSL)
- 3D Secure pour les cartes bancaires
- Passerelle de paiement certifiée PCI-DSS

Vos données bancaires ne sont jamais stockées sur nos serveurs.

## Puis-je payer en plusieurs fois ?

### Conditions
- Panier minimum : 100€ (3x) ou 200€ (4x)
- Carte bancaire valide
- Vérification instantanée

### Fonctionnement
1. Choisissez "Payer en 3x ou 4x" au checkout
2. Entrez vos informations carte
3. 1er prélèvement immédiat
4. Prélèvements suivants à 30/60/90 jours

### Frais
- **3x** : SANS FRAIS
- **4x** : SANS FRAIS

## Ma commande est refusée, pourquoi ?

### Causes fréquentes
1. **Fonds insuffisants** : Vérifiez votre solde
2. **Plafond atteint** : Contactez votre banque
3. **3D Secure échoué** : Vérifiez le code SMS
4. **Carte expirée** : Vérifiez la date de validité
5. **Adresse incorrecte** : Vérifiez l'adresse de facturation

### Solutions
- Réessayez avec une autre carte
- Contactez votre banque
- Utilisez PayPal

## Puis-je avoir une facture ?

### Facture automatique
- Envoyée par email à la validation de commande
- Disponible dans votre espace client

### Facture avec TVA récupérable
- Pour les professionnels
- Mentionnez votre numéro de TVA à l'inscription
- La facture sera émise avec TVA déductible

## Comment utiliser un code promo ?

1. Ajoutez vos articles au panier
2. Accédez au panier
3. Entrez le code dans le champ "Code promo"
4. Cliquez sur "Appliquer"
5. La réduction s'affiche

### Mon code ne fonctionne pas ?
- Vérifiez la date de validité
- Vérifiez le montant minimum
- Certains codes excluent les promotions
- Un seul code par commande

## Puis-je annuler une commande ?

### Avant expédition
- Contactez-nous immédiatement
- Annulation et remboursement complet

### Après expédition
- Refusez le colis à la livraison
- Ou suivez la procédure de retour
