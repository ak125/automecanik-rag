# Questions fréquentes - Suivi de commande

## Comment suivre ma commande ?

### Via votre espace client
1. Connectez-vous sur automecanik.com
2. Cliquez sur "Mon compte"
3. Accédez à "Mes commandes"
4. Le statut s'affiche en temps réel

### Via l'email de confirmation
- Cliquez sur le lien de suivi dans l'email d'expédition
- Vous serez redirigé vers le site du transporteur

## Quels sont les statuts de commande ?

| Statut | Signification |
|--------|---------------|
| **En attente de paiement** | Paiement non reçu ou en cours de validation |
| **Validée** | Paiement reçu, commande en préparation |
| **En préparation** | Les produits sont rassemblés |
| **Expédiée** | Colis remis au transporteur |
| **En livraison** | En cours d'acheminement |
| **Livrée** | Colis déposé/remis |

## Je n'ai pas reçu d'email de confirmation

### Vérifications
1. Consultez vos spams/courriers indésirables
2. Vérifiez l'adresse email de votre compte
3. Consultez votre espace client (la commande y apparaît)

### Si toujours rien
- Contactez le service client avec :
  - Votre nom
  - Date approximative de commande
  - Montant

## Ma commande est "en préparation" depuis longtemps

### Causes possibles
- **Produit sur commande** : Délai fournisseur
- **Rupture temporaire** : Réapprovisionnement en cours
- **Volume élevé** : Période de forte activité

### Que faire ?
- Consultez le délai indiqué sur la fiche produit
- Contactez le service client pour un point

## Le suivi n'est pas mis à jour

### Délai normal
- Mise à jour sous 24-48h après expédition
- Les week-ends ne sont pas comptés

### Statut bloqué ?
1. Attendez 48h supplémentaires
2. Contactez le transporteur avec le numéro de suivi
3. Si pas de réponse, contactez notre service client

## Mon colis est indiqué "livré" mais je n'ai rien reçu

### Vérifications immédiates
1. Vérifiez auprès de vos voisins
2. Regardez dans votre boîte aux lettres
3. Vérifiez auprès du gardien/concierge
4. Pour point relais : vérifiez le lieu indiqué

### Si introuvable
1. Contactez le transporteur sous 24h
2. Une enquête sera ouverte
3. Nous vous tiendrons informé

## Puis-je modifier ma commande ?

### Avant validation
- Modifiez directement votre panier

### Après validation, avant expédition
- Contactez-nous rapidement
- Modification possible selon l'avancement

### Après expédition
- Modification impossible
- Vous pouvez refuser le colis ou effectuer un retour

## Comment annuler ma commande ?

### Avant expédition
1. Connectez-vous à votre espace client
2. Accédez à "Mes commandes"
3. Cliquez sur "Annuler" (si disponible)
4. Ou contactez le service client

### Remboursement
- Sous 5 jours ouvrés après annulation confirmée
