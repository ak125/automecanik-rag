# Questions frequentes - Compte client

## Comment creer un compte?

### Inscription
1. Cliquez sur "Mon compte" en haut de page
2. Selectionnez "Creer un compte"
3. Remplissez le formulaire (email, mot de passe)
4. Validez votre email via le lien recu

### Avantages du compte
- Suivi de commandes
- Historique d'achats
- Factures telechargeables
- Enregistrement vehicules
- Adresses sauvegardees

## J'ai oublie mon mot de passe

### Reinitialisation
1. Cliquez sur "Mot de passe oublie"
2. Entrez votre adresse email
3. Recevez le lien de reinitialisation
4. Creez un nouveau mot de passe

### Lien non recu?
- Verifiez vos spams
- Verifiez l'adresse email utilisee
- Contactez le support si probleme persiste

## Modifier mes informations

### Dans votre espace client
- Adresse email
- Mot de passe
- Adresses de livraison/facturation
- Numero de telephone
- Vehicules enregistres

### Informations non modifiables
- Historique de commandes
- Factures emises

## Supprimer mon compte

### Procedure RGPD
1. Connectez-vous a votre compte
2. Parametres > Confidentialite
3. Demander la suppression
4. Confirmation par email

### Consequences
- Historique supprime definitivement
- Factures conservees (obligation legale 10 ans)
- Compte non recuperable

## Fusionner deux comptes?

### Non possible automatiquement
- Chaque compte est independant
- Historiques non transferables

### Solution
- Gardez le compte principal
- Supprimez le doublon
- Contactez-nous pour cas particuliers

## Compte professionnel

### Avantages
- Factures avec TVA detaillee
- Conditions specifiques (sur demande)
- Interlocuteur dedie

### Inscription
- Creez un compte standard
- Renseignez SIRET dans votre profil
- Statut pro active sous 24h
