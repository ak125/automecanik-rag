# Circuit de climatisation — Guide technique

## Composants
- Compresseur : pressurise le refrigerant gazeux
- Condenseur : evacue la chaleur, gaz → liquide
- Filtre deshydrateur : elimine humidite et impuretes
- Detendeur : regule le debit (valve thermostatique ou orifice fixe)
- Evaporateur : refroidit l'air habitacle par evaporation

## Circuit
- Haute pression (jaune/rouge) : entre compresseur et detendeur
- Basse pression (bleu) : entre detendeur et compresseur

## Entretien
- Inspection annuelle : controle visuel, test performance, filtre habitacle
- Tous les 2 ans : remplacement filtre deshydrateur, echange refrigerant, detection fuites
- A chaque ouverture du circuit : remplacer le filtre deshydrateur (hygroscopique)
- Joints toriques neufs, circuit ferme, quantites huile correctes.
