# Pompe a vide electrique — Guide technique

## Principe
Rotor excentrique avec palettes mobiles. La rotation cree des cavites qui s'agrandissent et se retrecissent, generant une depression pour alimenter le servo-frein.

## Applications
Vehicules ou la depression moteur est insuffisante : injection directe essence, diesel, hybrides, electriques, systemes Stop/Start.

## Symptomes
- Depression insuffisante dans le servo
- Pedale de frein dure
- Reduction des performances de freinage
- Voyant d'alerte (selon systeme)

## Diagnostic
Test fonctionnel : appuyer plusieurs fois sur la pedale de frein moteur tournant (>40 degC). La pompe doit s'activer audiblement et restaurer la depression. Diagnostic scanner pour surveiller la pression via capteur integre et lire les codes defaut.
