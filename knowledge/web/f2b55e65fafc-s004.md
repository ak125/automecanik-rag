### Outils utiles (lampe UV, talc, colorant)

L'aide d'une lampe UV constitue l'une des méthodes les plus efficaces pour localiser précisément une fuite. Pour l'utiliser :

Ajoutez un colorant fluorescent dans l'huile moteur (environ 7,5 ml par litre d'huile). Après avoir roulé suffisamment pour que le colorant circule dans tout le circuit, inspectez le moteur à l'aide d'une lumière UV. Les fuites apparaîtront avec une fluorescence jaune-verdâtre, rendant même les plus petites fuites facilement visibles.

Le talc est également un outil simple mais efficace. Appliqué sur les zones suspectées, il crée un contraste immédiat avec l'huile qui s'échappe. Pour les zones difficiles d'accès, le colorant fluorescent reste la meilleure solution, permettant de visualiser des fuites autrement invisibles à l'œil nu.
