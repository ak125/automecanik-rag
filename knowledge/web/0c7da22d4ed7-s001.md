# Comment changer les filtres des injecteurs d’essence

Les filtres situés à l’intérieur des injecteurs d’essence peuvent se boucher avec le temps, et lorsque cela se produit, le débit de carburant injecté dans le moteur est limité.

![Delphi - Diagnostics - 645 x 450px](_raw/web-images/a905918d0379048c.png)

En termes de conduite, cela pourrait signifier une réduction des performances ou une hausse de consommation de carburant, et si le carburant ne se mélange pas correctement, cela peut également entraîner une pollution plus élevée du moteur.
