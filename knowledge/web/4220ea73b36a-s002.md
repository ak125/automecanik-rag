## Pour une recherche précise

Pour pouvoir identifier l'article correspondant à un véhicule, vous devez indiquer, dans certains cas, le numéro de châssis ou le numéro PR du véhicule.

Les numéros de châssis sont gravés dans le compartiment moteur, au niveau du seuil de porte à côté du siège passager avant et dans le coffre.

![](_raw/web-images/90101b583c5aa93d.jpg)

Les modèles de véhicules pour lesquels l'article est disponible sont indiqués dans cette colonne avec leurs numéros de châssis respectifs :

![](_raw/web-images/1e0a59143acae869.jpg)

Les numéros de châssis d'Audi, Seat et Skoda sont composés d'une combinaison de 17 chiffres et lettres ; le catalogue indique les onze derniers caractères.

![](_raw/web-images/f356538630302239.jpg)

Les numéros de châssis d'Opel sont composés d'une combinaison de 17 chiffres et lettres. Vous aurez besoin des huit derniers caractères.

Les véhicules VW, Audi, Seat et Skoda possèdent également un numéro appelé « numéro PR » qui permet une identification précise.

Le fabricant parle d'un « support de données pour le service client » : il s'agit d'un autocollant qui se trouve dans le coffre et/ou le carnet d’entretien.

Dans notre exemple, les deux flèches indiquent les numéros PR du système de freinage utilisé.

![](_raw/web-images/5aabc2a9184104ec.jpg)

Vous trouverez le numéro PR dans cette colonne du catalogue.
