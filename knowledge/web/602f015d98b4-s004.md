### Un outil d’écoute sans fil peut-il être utilisé pour diagnostiquer d’autres dysfonctionnents ?

Oui. Vous pouvez utiliser l’outil pour identifier les défauts du moteur, des essieux et du châssis d’un véhicule. Idéal pour diagnostiquer les injecteurs de carburant, les roulements, les engrenages, les suspensions, les transmissions, les différentiels et les supports d’essieu défectueux.
