### Comment nettoyer les étriers de frein :

Pour obtenir les meilleurs résultats, le nettoyage de l’étrier de frein nécessitera probablement un démontage. Mais il existe une autre méthode pour nettoyer les étriers de frein sans démonter les pneus. Utilisez une petite brosse avec un peu de nettoyant pour freins, brossez et rincez ensuite. Certains nettoyants pour freins sont conçus pour être rincés et ainsi éliminer la poussière. Vous pouvez également faire un petit tour avec le nettoyant. Cela aidera le nettoyant à pénétrer dans les fissures.
