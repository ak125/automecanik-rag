## Mais quelle est la cause d’une usure prématurée des plaquettes de frein ?

Un certain nombre de conditions et de styles de conduite peuvent accélérer le processus normal d’usure des plaquettes de frein, et les facteurs les plus courants, ainsi que leurs conséquences, sont reportés ci-dessous.

![Contacter le support Brembo](_raw/web-images/1e8d4f4c405ed310.webp)

| FACTEURS | CONSEQUENCES |
| --- | --- |
| Conduite agressive | Usure rapide du matériau |
| Conduite en ville/sur routes de montagne | Usure accrue |
| Disques endommagés | Usure irrégulière |
| Matériaux de mauvaise qualité | Sécurité inférieure |
| Montage incorrect | Usure irrégulière du matériau |
| Dysfonctionnement des étriers | Usure irrégulière |
| Matériau agressif | Détérioration accélérée |
| Prévention par |  |
| l’utilisation de pièces détachées correctes et un entretien régulier |  |
