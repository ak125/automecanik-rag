# Reservoir a charbon actif (EVAP) — Guide technique

## Fonction
Stocke les vapeurs de carburant evaporees du reservoir. La vanne de purge libere les vapeurs stockees vers l'admission quand la regulation lambda est active — combustion des vapeurs dans le moteur.

## Fonctionnement
La depression du collecteur d'admission aspire l'air ambiant a travers le filtre charbon, entrainant les vapeurs stockees vers le moteur.

## Codes defaut
- P0441 : debit de purge incorrect
- P0440-P0457 : famille codes systeme EVAP

## Diagnostic
Inspecter dommages physiques, verifier integrite durites, connexions electriques, mesurer tension (11-14V moteur chaud), oscilloscope pour signaux vanne de purge. Regulation lambda active obligatoire pour test valide.
