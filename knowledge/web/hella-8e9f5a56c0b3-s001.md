# Feux de signalisation — Guide technique

## Types et intensites
| Feu | Couleur | Intensite |
|---|---|---|
| Feux stop (S1) | Rouge | min 60 cd |
| Feux position (R) | Rouge | 4 cd |
| Clignotants | Orange | 60-120 flashs/min |
| Antibrouillard (F) | Rouge | surface max 140 cm2 |
| Recul (AR) | Blanc | 1-2 unites |
| Eclaireur plaque | Blanc | min 2,5 cd/m2 |

## Montage
Hauteur : 350-1500 mm, espacement lateral min 600 mm

## Diagnostic
Verifier : etat ampoule, corrosion connecteurs, fusibles, signaux PWM a l'oscilloscope sur vehicules recents.
