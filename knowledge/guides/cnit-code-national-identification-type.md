Le CNIT est le code alphanumérique officiel qui identifie un véhicule en France. Indispensable pour l'immatriculation et la commande de pièces détachées, il remplace l'ancien type Mine depuis 1995.

## Définition et historique du CNIT

Le Code National d'Identification du Type (CNIT) est un identifiant alphanumérique attribué à chaque véhicule réceptionné en France. Mis en place progressivement à partir de 1995, il a officiellement remplacé le type Mine en 1998.

Le type Mine était l'ancien système d'identification, encore mentionné dans le langage courant. L'appellation officielle est désormais « Code National d'Identification du Type ». Le CNIT a été conçu pour s'aligner sur les réglementations européennes et simplifier l'identification des véhicules au sein de l'Union Européenne.

Ce code atteste que le véhicule a bénéficié d'une réception de conformité européenne (réception CE). Il est requis par l'ANTS (Agence Nationale des Titres Sécurisés) pour toute demande de certificat d'immatriculation.

## Structure du CNIT : 12 ou 15 caractères

La longueur du CNIT dépend de la date de première immatriculation du véhicule :

**12 caractères** — véhicules immatriculés entre 1996 et 2009 (ancien système FNI).

**15 caractères** — véhicules immatriculés depuis le 15 avril 2009 pour les neufs, et le 15 octobre 2009 pour les occasions (système SIV). Le numéro d'immatriculation SIV est attribué à vie : le véhicule le conserve jusqu'à son exportation ou sa destruction.

### Signification des caractères clés

| Position | Information |
|----------|------------|
| 4e caractère | Transmission et type d'énergie (essence, diesel, électrique, hybride) |
| 5e caractère | Tranche de puissance fiscale |
| 7e caractère | Type de carrosserie (berline, break, SUV, utilitaire…) |

Ces informations sont codifiées sur une échelle allant de 0 à 7 ou de 0 à 9 selon les positions.

### Lien avec le TVV (Type-Variante-Version)

Le CNIT est étroitement lié au TVV, un code alphanumérique défini par les constructeurs automobiles. Le TVV précise le modèle, la marque, le genre et le niveau d'équipement du véhicule. Il est requis dans le cadre de la réception CE, qui atteste que le véhicule respecte les normes techniques européennes.

Le numéro de réception communautaire, inscrit sur la carte grise, garantit la conformité du véhicule dans tous les pays membres de l'Union Européenne.

## Où trouver le CNIT sur la carte grise

Le CNIT se trouve dans le **champ D.2.1** du certificat d'immatriculation (carte grise au format européen).

### Autres champs utiles de la carte grise

| Champ | Information |
|-------|------------|
| **A** | Numéro d'immatriculation |
| **B** | Date de première immatriculation |
| **D.1** | Marque du véhicule |
| **D.2** | Type, variante, version (TVV) |
| **D.2.1** | **CNIT** — Code National d'Identification du Type |
| **D.3** | Dénomination commerciale |
| **E** | Numéro d'identification du véhicule (VIN) |
| **F.1** | Masse en charge maximale (PTAC) |
| **J** | Catégorie du véhicule (VP, CTTE…) |
| **J.1** | Genre national (VP, CTTE, TCP…) |
| **P.1** | Cylindrée en cm³ |
| **P.3** | Type de carburant / énergie |
| **P.6** | Puissance administrative (CV fiscaux) |

Le numéro de formule du certificat d'immatriculation (pour les véhicules mis en circulation après 2004) se trouve sur la couverture du document.

## CNIT et véhicules importés

Le CNIT est un code propre à la France. Il n'apparaît pas systématiquement sur les certificats d'immatriculation étrangers, y compris ceux émis dans des pays membres de l'UE (Allemagne, Espagne, Italie, Belgique…).

Pour un véhicule importé, le CNIT est reconstitué par les organismes d'immatriculation français à partir des informations présentes sur la carte grise étrangère, notamment le TVV (Type-Variante-Version).

Si vous importez un véhicule d'occasion d'un pays de l'UE, le recours à un professionnel spécialisé dans l'importation automobile facilite la démarche d'immatriculation et la détermination du CNIT — un identifiant également nécessaire pour commander des pièces détachées compatibles.

## CNIT et commande de pièces détachées

Le CNIT est une information précieuse pour identifier avec précision un véhicule et commander les bonnes pièces détachées. Sur AutoMecanik, plusieurs méthodes d'identification sont disponibles :

1. **Par plaque d'immatriculation** (méthode la plus rapide) — le système identifie automatiquement le véhicule à partir de la plaque SIV ou FNI.
2. **Par numéro VIN** (17 caractères) — identification universelle, y compris pour les véhicules importés.
3. **Par sélection manuelle** — marque, modèle, motorisation, année.
4. **Par référence OEM** — si vous connaissez déjà la référence constructeur de la pièce.

Le CNIT, combiné au VIN et aux informations de la carte grise (champs D.1, D.2, P.3), permet de lever toute ambiguïté sur les variantes de montage et de garantir la compatibilité des pièces commandées parmi les 4 millions de références du catalogue AutoMecanik.

### Champs carte grise essentiels pour commander des pièces

- **D.2.1 (CNIT)** : identifie le type exact du véhicule
- **E (VIN)** : numéro de châssis unique à 17 caractères
- **P.3** : type d'énergie (essence, diesel, GPL, électrique)
- **D.3** : dénomination commerciale (ex : "308 II 1.6 BlueHDi 120")
